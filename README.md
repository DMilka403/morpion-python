# Morpion en Python

Jeu du morpion (tic-tac-toe) développé en Python dans le cadre de projets et d'entraînement en programmation. (NSI)

## Fonctionnalités
- Jeu à 2 joueurs
- Jeu contre une IA
- IA avec logique sur grille 3x3
- Taille de grille paramétrable
- Affichage clair de la grille dans le terminal

## Lancer le jeu
```bash
python main.py
```

## Règles
- Le joueur 1 joue avec le symbole X
- Le joueur 2 (ou l'IA) joue avec le symbole O
- Le premier à aligner ses symboles gagne

## Technologies
- Python 3

## Remarques
Projet réalisé pour pratiquer :
- la décomposition en fonctions
- la gestion de structures de données
- la logique algorithmique
- l'analyse et le test du code
